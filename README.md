# websocketproxy-GO
Reserve proxy websocket in GO 

# FAST PROXY STABLE HTTP

# INSTALLATION

```
apt install golang-go
```

```
wget https://raw.githubusercontent.com/MAHBOUB-MILLON/websocketproxy-GO/main/proxy.go
```

```
chmod +x proxy.go
```

```
go run proxy.go
```
